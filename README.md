# WTC – Smart Water Level Controller

WTC is an ESP8266-based smart water controller and monitoring system designed to automate water tank management using IoT technology.

This project includes:

- ESP8266 firmware
- Real-time water level monitoring
- Automatic motor control
- Web dashboard using HTML, CSS and JavaScript
- Firebase cloud integration
- Responsive UI for mobile and desktop

---

## Features

- Real-time water level monitoring
- Automatic motor ON/OFF control
- Manual motor control from web dashboard
- Tank percentage indicator
- Firebase realtime database integration
- Mobile responsive UI
- Device online/offline status
- Live data updates without refresh
- ESP8266 WiFi connectivity
- Overflow and dry-run protection support

---

## Technologies Used

### Hardware

- ESP8266 NodeMCU
- Ultrasonic Sensor
- Relay Module
- Buzzer
- Float Sensor (optional)

### Software/Tech

- HTML5
- CSS3
- JavaScript
- Firebase Realtime Database
- ESP8266 Arduino Framework
